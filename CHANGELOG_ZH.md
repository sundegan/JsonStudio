# 变更记录

此文件用于记录项目的主要变更。

## [0.5.0] - 2026-03-04

### 新增
- 图片导出功能，基于 Rust 原生渲染，支持水印和剪贴板复制。
- JSON Schema 生成与验证。
- JSON 数据生成多种编程语言的类/结构体代码及反向转换。
- JSON 与 YAML、XML、TOML、CSV 双向格式转换，CSV 支持彩虹列高亮。
- JSON 数据一键修复。
- 全面中英文国际化，设置面板新增语言选择。
- 编辑器快捷键可在设置面板中自定义配置。
- 新增 Rose Ivy 浅色主题。

### 改进
- 工具栏重新布局以及图标优化。
- 树视图布局和响应式表现优化。
- Diff 编辑器样式优化，移除独立 DiffTabBar 组件。

### 修复
- 修复冷启动时文件关联打开显示空白的竞态条件问题。
- 修复大内容图片导出空白/模糊的问题。

### 移除
- 移除 Diff 模式快捷键。
- 移除 Quiet Light 主题（由 Rose Ivy 替代）。

## [0.4.0] - 2026-02-28

### 新增
- **JMESPath 查询支持**：用 JMESPath 表达式替代 Tree View 中的简单文本过滤，支持强大的 JSON 查询能力，实时高亮匹配节点并自动展开祖先路径。
- **新增配色主题**：新增 Dracula、Nord、Solarized Light、Quiet Light、Catppuccin Latte 五款编辑器主题。
- **JSON 文件关联**：在系统中注册 `.json` 文件类型，支持双击直接用 Json Studio 打开 JSON 文件。
- **Tree View 关闭按钮**：新增关闭按钮，可快速隐藏 Tree View 面板。
- **JMESPath 帮助弹窗**：内置快速参考文档，包含示例数据和常用查询模式。

### 改进
- **复制路径 + 值**：Tree View 复制按钮现在同时复制 JMESPath 路径和值（如 `users[0].name: "Alice"`），而不是仅复制 JSON Pointer 路径。
- **展开/折叠切换**：将展开全部和折叠全部合并为一个切换按钮，图标随状态动态变化。
- **JSON 错误提示**：显示更清晰的警告图标和 "Invalid JSON Format" 提示，替代纯文本错误信息。
- **设置面板重构**：优化布局结构和样式，提升可读性和可维护性。
- **Tree View 布局优化**：改善工具栏间距、对齐方式，提升不同面板宽度下的响应表现。
- **CSS 架构优化**：引入 JSON 数据类型颜色 CSS 变量（object、array、string、number、boolean、null），统一主题配色。

### 修复
- 修复查找组件的 hover 事件冒泡导致 Monaco 编辑器出现意外行为的问题。
- 修复切换标签页时编辑器内容变更未正确保存的问题。

### 移除
- 移除独立的 `JsonQueryPanel` 组件（功能已由 Tree View 中的 JMESPath 查询替代）。

## [0.3.0] - 2026-01-13

### 新增
- **多标签支持**：实现标签管理功能，支持拖拽排序。
- **标签右键菜单**：包含固定标签、关闭其他标签、关闭所有标签等选项。
- **固定标签**：可固定重要标签，防止误关闭。
- **JSON树形视图**：在编辑器旁显示 JSON 结构树，支持展开/折叠、搜索过滤、点击定位功能。
- **JSON查询面板**：使用JSONPath或JSON Pointer表达式查询JSON数据。
- **对比模式支持多标签**：对比模式下左右两侧独立的标签管理。
- **行高设置**：允许用户手动调整编辑器行高。

### 修复
- 修复对比模式下切换标签页时编辑器内容不更新的问题。
- 修复固定标签页时标签页变大的问题，将固定图标移至左侧区域。

### 改进
- Tree View 侧边栏现在占据工具栏下方的完整高度（与标签栏和编辑器区域并列）。
- 状态栏在对比模式下高亮显示差异行数。
- 调整状态栏字体大小（13px）并减小高度，使界面更紧凑。
- 压缩 Tree View 头部区域，优化空间利用。
- 统一普通模式和对比模式下的标签栏高度（24px）和样式。
- 更新工具栏新建文件图标和标签固定图标设计。
- 禁用编辑器浏览器右键菜单，提供更清爽的体验。
- 模块化 JsonEditor 组件（工具栏、查询面板、状态栏、提示），提升可维护性。

## [0.2.0] - 2026-01-11

- 粘贴自动格式化。
- Makefile：简化开发与构建流程。
- 变更记录文档：补充版本更新说明。

### 改进
- 剪贴板处理与快捷键行为。
- JSON 格式化体验与 Monaco 编辑器集成。
- 快捷键触发下的窗口管理行为。

## [0.1.0] - 2026-01-07

### 新增
- 初始版本发布。
- JSON 格式化与压缩。
- 转义/反转义工具。
