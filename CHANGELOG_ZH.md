# 变更记录

此文件用于记录项目的主要变更。

## [0.2.0] - 2026-01-11

### 新增
- 文件操作：打开/保存/新建/文件拖拽。
- Diff 模式：在编辑器内进行 JSON 对比。
- 粘贴自动格式化。
- Makefile：简化开发与构建流程。
- 变更记录文档：补充版本更新说明。

### 改进
- 剪贴板处理与快捷键行为。
- JSON 格式化体验与 Monaco 编辑器集成。
- 快捷键触发下的窗口管理行为。

## [0.1.0] - 2026-01-07

### 新增
- 初始版本发布。
- JSON 格式化与压缩。
- 转义/反转义工具。
- JSON 统计（键数量/层级/大小）与校验。
- 基于 Monaco 的 JSON 编辑器：语法高亮、折叠、行号。
- 剪贴板集成与全局快捷键。
- 主题切换（明/暗）与编辑器设置（字体大小、缩进），以及快捷键设置。
