如何在 JsonStudio 中使用 JSON 格式化工具

  1. 下载并安装 JsonStudio，访问 GitHub Releases（支持 macOS、Windows 和 Linux）
  2. 启动应用 – 不到 1 秒即可打开
  3. 粘贴您的 JSON 数据 – 粘贴时自动格式化
  4. 或按 Ctrl+Shift+F（Windows/Linux）或 Cmd+Shift+F（macOS）手动格式化
  5. 复制格式化后的结果 – 或使用 Ctrl+S 保存到文件
