如何在 JsonStudio 中使用 JSON 验证工具

根据 Schema 验证 JSON

  1. 下载并安装 JsonStudio，访问 GitHub Releases
  2. 从左侧边栏打开 Schema 工具
  3. 在左侧窗格粘贴您的 JSON 数据
  4. 在右侧窗格粘贴您的 JSON Schema
  5. 查看验证结果 – 错误会高亮显示并提供详细消息

从数据生成 JSON Schema

  1. 从左侧边栏打开 Schema 工具
  2. 在左侧窗格粘贴您的 JSON 数据
  3. 点击"生成 Schema"按钮
  4. Schema 出现在右侧窗格，可以直接使用或自定义
