什么时候使用哪种视图？

  • 树视图：适合深层嵌套对象、混合数组、路径查看和结构调整。
  • Grid 网格视图：适合用户、订单、事件、接口记录等结构相似的对象数组。
  • 源码编辑器：适合批量文本编辑、格式化、折叠和精确语法操作。

典型操作流程

  1. 在 JsonStudio 中粘贴或打开 JSON。
  2. 打开树视图检查嵌套结构，并编辑具体的 key 或 value。
  3. 需要调整结构或顺序时，拖拽支持的树节点。
  4. 根数据是记录数组时，切换到 Grid 网格视图。
  5. 编辑单元格，然后返回源码编辑器继续处理，同步修改会完整保留。
