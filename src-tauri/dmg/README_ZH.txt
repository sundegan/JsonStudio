Json Studio macOS 安装说明
==========================

1. 打开这个 DMG 文件。
2. 将 "Json Studio.app" 拖入 "Applications" 文件夹。
3. 从“应用程序”中打开 Json Studio。

macOS 首次打开提示
------------------

当前 macOS 安装包未使用 Apple Developer ID 公证。因此首次打开时，
macOS 可能会提示应用来自未识别开发者，或阻止直接打开。

如果 macOS 阻止打开：

1. 打开 Finder，进入“应用程序”。
2. 右键点击 "Json Studio.app"，选择“打开”。
3. 在安全提示中再次点击“打开”。

如果没有看到“打开”按钮，请进入：

系统设置 > 隐私与安全性

找到 Json Studio 相关提示，并允许打开。

说明
----

- Json Studio 在你的电脑本地运行。
- 这个 DMG 不包含安装脚本。
- 正常安装不需要执行 Terminal 命令。
