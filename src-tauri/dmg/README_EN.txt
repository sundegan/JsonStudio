Json Studio macOS Installation Guide
====================================

1. Open this DMG file.
2. Drag "Json Studio.app" into the "Applications" folder.
3. Open Json Studio from Applications.

First launch on macOS
---------------------

Current macOS builds are not notarized with Apple Developer ID. Because of that,
macOS may show a security warning the first time you open Json Studio.

If macOS blocks the app:

1. Open Finder and go to Applications.
2. Right-click "Json Studio.app" and choose "Open".
3. Click "Open" again in the security dialog.

If the "Open" button is not shown, go to:

System Settings > Privacy & Security

Then find the Json Studio warning and choose to allow opening it.

Notes
-----

- Json Studio runs locally on your computer.
- No install script is included in this DMG.
- No Terminal command is required for normal installation.
